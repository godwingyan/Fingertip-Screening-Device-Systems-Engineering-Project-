# Smart Fingertip Screening Device

## Overview

The Smart Fingertip Screening Device is an embedded health-monitoring system designed to measure fingertip temperature in real time while providing visual feedback for proper hand placement. The project was developed as a systems engineering case study integrating hardware, software, requirements analysis, and SysML modeling.

The device detects hand presence using an ultrasonic sensor, measures temperature using a thermistor, and displays results through a 4-digit LED display with red/green LED indicators. MQTT communication enables real-time data transmission for monitoring and visualization.

---

## Problem Statement

Traditional temperature screening systems can be expensive, slow, or inconvenient in environments requiring rapid screening. This project explores a low-cost embedded solution capable of providing fast fingertip temperature readings while minimizing user error through feedback mechanisms.

---

## System Features

* Real-time fingertip temperature measurement
* Hand proximity detection using ultrasonic sensing
* LED-based positioning guidance
* 4-digit temperature display
* MQTT communication for remote monitoring
* Embedded control using Raspberry Pi Pico
